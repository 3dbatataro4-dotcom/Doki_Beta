遊戲企劃書：《紙校：虛妄的迴響》
Game Design Document (GDD)
1. 遊戲概述 (Game Overview)
* 遊戲名稱：紙校：虛妄的迴響 (Paper School: Echoes of the Absent)
* 平台：HTML5 (Mobile Vertical / 直式手機網頁)
* 類型：乙女戀愛模擬 (Otome) / 心理恐怖 (Psychological Horror) / Meta Game
* 核心體驗：
   * 表象：治癒、校園戀愛、填補失戀少年的心。
   * 真相：角色因思念過度而崩壞，玩家只是「代餐」，遊戲程式本身會隨著角色的精神狀態崩潰。
* 技術棧：HTML, CSS, JavaScript (Vanilla or simple framework like Ren'JS logic implementation).

2. 故事背景 (Story Setting)
玩家轉學來到「紙校」（Paper High School），加入了一個名為**「身心靈自助研究社」**的奇怪社團。社團裡只有三名核心成員（加上玩家共四人）。 這三位風雲人物在上個月同時遭遇了情感重創——他們各自的「靈魂伴侶」（林恩、喬諾娜、蜜拉思）集體轉學消失了。 為了防止自己發瘋，他們聚在一起試圖通過「社團活動」來轉移注意力。玩家的任務是陪伴他們，度過這段艱難時期。
3. 角色詳情與AI指令 (Character Profiles & AI Instructions)

A. 彼得 (Peter)
* 人設：三年級學長。自帶王者氣場但精神不穩定的學生會成員。
* 性格關鍵詞：責任感、強迫症、容易中邪、迷信、對陌生人冷漠（口頭禪：Shit）、認為潔廁靈能驅邪。
* 思念對象：林恩 (Lynn) 。
* 普通模式表現：
   * 會隨身攜帶聖水或奇怪的清潔劑。
   * 對玩家態度冷淡，稱玩家為「庶民」或「新來的」。
   * 如果玩家選擇「吉祥」的選項，好感度會上升。
* 崩壞模式表現 (Week 2)：
   * 中邪：對話框文字會變成亂碼（Zalgo text）。
   * 幻覺：會對著空氣說話，把玩家看成林恩的殘影。
   * 行為：強迫玩家喝「藍色聖水」（潔廁靈），或者在背景中看到無數個「Yellow Shit」（納希瑟斯）在嘲笑他。
   
B. 蘭蘭 (Lanlan / Virdrakos)
* 人設：二年級學弟。體育特長生，非常有錢的混血兒。
* 性格關鍵詞：開朗、沒品、道德觀低下、怕鬼、喜歡火、對朋友大方。句尾偶爾加「呀」。
* 思念對象：喬諾娜 (Jornona) 。
* 普通模式表現：
   * 陽光開朗，會說「給你看個寶貝」然後掏出鑽石或打火機。
   * 如果玩家表現得像喬諾娜（軟、順從），他會很開心。
   * 很怕社團活動室晚上的陰影。
* 崩壞模式表現 (Week 2)：
   * 破壞慾：背景圖片會出現「燒焦」的痕跡。
   * 病嬌：笑著說要為了保護玩家把學校燒了。
   * 認知錯亂：會哭著抱住玩家喊「喬諾娜」，然後瞬間變臉說「你不是他，燒了吧。」
   
   
C. 奧拉 (Ora)
* 人設：數學老師兼社團指導老師（或三年級特優生，視校園Paro設定，此處設定為高冷的天才社長以方便戀愛）。
* 性格關鍵詞：高智商、高自尊、小心眼、話少、偏執、掌控欲。
* 思念對象：蜜拉思 (Melas) 。
* 普通模式表現：
   * 冷靜理智，負責制定社團規則。
   * 討厭吵鬧，喜歡安靜和觀察。
   * 會記住玩家選錯的每一個選項（小心眼）。
* 崩壞模式表現 (Week 2)：
   * Meta控制：他意識到這是個遊戲。他會「暫停」背景音樂。
   * 時間操作：強制將時間倒流，讓玩家重複經歷痛苦的對話。
   * 冷酷實驗：把玩家當作實驗品，試圖通過玩家找回蜜拉思的數據。

4. 遊戲循環與玩法機制 (Game Loop & Mechanics)
每日流程 (Daily Cycle)
每天有 4個行動點 (Action Points)：
1. 早晨上課 (Morning Class)：選擇坐在誰旁邊（增加少量好感）。
2. 課間聊天 (Break Chat)：觸發短對話事件。
3. 午休約會 (Lunch Date)：選擇和誰一起吃飯（重要好感分歧）。
4. 社團活動 (Club Activity)：進行小遊戲（核心玩法）。
5. 放學 (After School)：是否一起回家。
6. 晚間 (Night)：SNS 互動 (類似LINE/IG的介面)。
社團小遊戲：思維拼圖 (Mind Puzzle)
類似 Doki Doki Literature Club 的寫詩系統，但改為「挑選關鍵詞來完成社團報告」。 屏幕上會出現隨機單詞，玩家需選擇單詞，不同角色對不同單詞有反應（Q版小人會跳動）。
* 彼得喜歡的詞：吉祥、貓、乾淨、王、驅邪、廁所、閃亮。
* 蘭蘭喜歡的詞：火、寶石、燒烤、老婆、玩樂、有錢、櫻桃。
* 奧拉喜歡的詞：時間、觀察、實驗、安靜、邏輯、有趣、有毒。
週末約會 (Weekend)
* 第五天結束後觸發。好感度最高的角色會邀請玩家去特殊地點。
   * 彼得：教堂。
   * 蘭蘭：遊樂園。
   * 奧拉：天文館。

5. 劇本進程與恐怖轉折 (Script Progression)
第一周：虛假的平靜 (Week 1: The False Peace)
* 氛圍：明亮、UI乾淨、BGM輕鬆。
* 劇情：玩家剛入社，大家雖然偶爾提到「這間教室以前更熱鬧（指前任還在時）」，但整體還是歡迎玩家的。
* SNS：角色會發布一些風景照，配文略帶憂鬱但正常。例如蘭蘭發一張火堆照：「想念那時候烤的棉花糖呀。」

第二周：崩壞開始 (Week 2: The Glitch)
* 觸發點：週末約會結束後，玩家發現角色的狀態欄出現亂碼。
* 彼得的變化：
   * 上課時會突然站起來大叫「有鬼！」。
   * 對話框的文字會溢出邊界。
   * 如果不選他，他會發送空白簡訊騷擾玩家。
* 蘭蘭的變化：
   * 立繪的眼睛偶爾會變成獸瞳（豎瞳）。
   * 小遊戲中，無論選什麼詞，他都說是「想燒掉的東西」。
   * 會問玩家：「你的皮膚...撕下來會痛嗎？喬諾娜就不會痛（因為是仿生人）。」
* 奧拉的變化：
   * 直接刪除其他角色的選項。
   * 對話中打破第四面牆：「你以為重新讀檔我就不記得了嗎？」
   * 背景音樂會變調、倒放。
   
*具體改變根據遊戲路線內容決定。


第三周：結局 (Week 3: The Endings)

6. 特殊玩法：白月光歸來 (The "White Moonlight" Mechanic)
這是本遊戲的核心亮點。在遊戲開始輸入玩家姓名時，會進行字串檢測。
情況 A：玩家輸入 "林恩" (Lynn)
* 彼得反應：
   * 狀態：瞬間變成「忠犬/昏君」模式 。
   * 劇情：不再有恐怖元素。彼得會瘋狂撒嬌，叫玩家「貓貓」，把所有資產（遊戲道具）都送給玩家。

情況 B：玩家輸入 "喬諾娜" (Jornona)
* 蘭蘭反應：
   * 狀態：瞬間變成「戀愛腦奶狗」模式 。
   * 劇情：蘭蘭會一直黏著玩家，一直說「喬諾娜～」，背景會飄櫻桃和愛心。任何恐怖選項都被強制替換成「抱抱」。

情況 C：玩家輸入 "蜜拉思" (Melas)
* 奧拉反應：
   * 狀態：從高冷變成「感興趣/被拿捏」模式 。
   * 劇情：奧拉會主動靠近，嘴角上揚。他會說：「你終於玩夠回來了嗎？」恐怖的Meta能力變成「為了你修改遊戲數據」。

*白月光模式中，若與非CP人物互動會引發對應路線男主吃醋。（例如玩家名稱叫林恩時去找奧拉互動，會引發彼得吃醋。）

2. 初始狀態 (Common Route - Week 1)
前置劇情：玩家剛加入社團，彼得坐在角落，桌上擺著一瓶藍色的清潔劑，神情緊繃。
初次見面 (First Encounter)
* 場景：社團教室（黃昏）
* 彼得：（眼神陰鷙，盯著空氣中的某個點）這裡的風水還是不對...那邊角落有個髒東西在看我。
* 玩家：你好，我是新來的社員...
* 彼得：（瞥了一眼）嘖。又是個毫無靈力的庶民。離我遠點，別把你的晦氣傳染給我。
* 彼得：（拿出一瓶寫著『強效去污』的藍色液體喝了一口）呼...Shit，終於壓下去了。
好感度機制 (Affection Mechanics)
在第一週，彼得對玩家的態度取決於**「是否乾淨/吉祥」**。
* 喜歡的詞彙（小遊戲）：乾淨、驅邪、王、貓、錢、聖水、秩序。
* 討厭的詞彙：髒、鬼、混亂、鏡子（會聯想到納希瑟斯/Yellow Shit ）。

3. 分支 A：普通/恐怖路線 (Normal/Horror Route)
觸發條件：玩家名稱 不包含 「林恩」、「Lynn」。
第二周：崩壞 (The Glitch)
彼得因為過度思念林恩，壓制不住體內的「純陰體質」，開始頻繁中邪 。他將玩家視為導致風水變差的「不潔之源」。
* 異常行為 1 (幻覺)：
   * 上課時，彼得會突然對著玩家大吼：「你背後趴著什麼？！Shit！它在笑！」
   * 對話框中的文字會開始出現亂碼，例如：「你...是...穢物...Shit...」
* 異常行為 2 (強迫症)：
   * 彼得會瘋狂地擦拭桌子，直到桌子被磨穿。
   * 他會試圖強迫玩家喝下他的「聖水」（潔廁靈）來驅邪。
關鍵劇情：午夜的驅魔 (Midnight Exorcism)
* 場景：深夜的社團教室，只有彼得和玩家。
* 彼得：（戴著面具，手持馬桶刷，眼神空洞）林恩不在...貓貓不在...這裡太髒了。
* 彼得：是你吧？把我的吉祥運氣吸走的人。
* 彼得：只要把你洗乾淨...林恩就會回來了。
* 恐怖演出：
   * 螢幕上出現大量紅色的 "CLEAN IT" (洗乾淨)。
   * 背景音樂變成失真的聖歌與沖水聲的混合。
   * 彼得的立繪頭部會偶爾以不自然的角度扭曲。

4. 分支 B：白月光路線 (True Love/Sweet Route)
觸發條件：玩家名稱輸入為 「林恩」 或 「Lynn」。
特殊開場 (Special Intro)
當玩家輸入名字確認的瞬間，UI 介面從陰暗色調瞬間變成金碧輝煌的風格。
* 彼得：（瞳孔地震，手中的潔廁靈掉在地上）......
* 彼得：（衝過來，甚至有點踉蹌）林恩？你是林恩？
* 彼得：Shit...我以為我再也見不到你了！我的貓！我的藥！
態度轉變 (Attitude Shift)
* 稱呼變化：從「庶民」變為 「貓貓」、「創意寶寶」、「林恩」 。
* 性格變化：
   * 忠犬/昏君模式：雖然還是那副高高在上的國王語氣，但對玩家百依百順。「只要你待在我身邊，要多少資產都給你。」
   * 無中邪：只要玩家在場，彼得完全不會中邪，且精神極度穩定 。
   * 醋罈子：極度排斥其他角色接近玩家。
專屬甜寵劇情 (Sweet Events)
事件一：資產轉移 (The Assets)
* 場景：午休時間。
* 彼得：這是學生會的印章，這是社團的經費卡，這是我名下公司的股權讓渡書。
* 玩家 (林恩)：...這太多了，彼得。
* 彼得：收下。放在你那裡比較吉祥。我這種容易招鬼的體質，錢放在我這只會變成冥幣。你是我的保險箱。
事件二：驅趕閒雜人等 (Jealousy)
當 蘭蘭 (Lanlan) 試圖靠近玩家時：
* 蘭蘭：哎呀，林恩回來了？借我點錢買個戒指給喬諾娜呀...
* 彼得：（瞬間拔出馬桶刷/權杖擋在中間）滾開，你這個一身煙火味的蜥蜴。
* 彼得：別靠近我的貓。你的火會燒壞他的財運。去跟你的那個粉紅色仿生人玩去。
當 納希瑟斯 (Narcissus) 的照片出現或被提及其名字時：
* 彼得：（一臉嫌棄）Yellow Shit。
* 玩家 (林恩)：我覺得納希瑟斯挺好看的啊...
* 彼得：（強忍怒氣，深呼吸）...既然貓貓這麼說。行吧。但我還是要在他的照片上貼符咒。

2. 初始狀態 (Common Route - Week 1)
前置劇情：蘭蘭通常坐在課桌上（而不是椅子上），手裡玩著一枚昂貴的鑽戒或是防風打火機。
初次見面
* 場景：社團教室（充滿陽光）。
* 蘭蘭：（瞇著眼睛笑，露出小虎牙）新來的？我是蘭蘭呀！
* 蘭蘭：彼得說社團要招人，不然又要中邪了。你也是來避邪的嗎？
* 玩家：我是來加入社團的...
* 蘭蘭：那正好呀！我們來玩吧！彼得整天神神叨叨的，奧拉又不講話，蘭蘭好無聊呀。
* 蘭蘭：（突然湊近）對了，你有看到鬼嗎？這裡沒有鬼吧？蘭蘭不喜歡那個呀！
好感度機制
* 喜歡的詞彙：寶石、錢、玩樂、喬諾娜、老婆、開心、燒烤。
* 討厭的詞彙：鬼、幽靈、學習、沒錢、規矩、冷。
* 特殊互動：如果玩家選擇「講鬼故事」，蘭蘭好感度會暴跌，並且可能會直接掏出打火機把故事書燒了 。

3. 分支 A：普通/恐怖路線 (Normal/Horror Route)
觸發條件：玩家名稱 不包含 「喬諾娜」、「Jornona」、「蘇郎」(外貌相似)。
核心概念：失控的道德觀 (The Amoral Dragon)
